# Emotion-Aware Music Recommendation Engine

An intelligent music recommendation system that analyzes user emotions through facial recognition and text sentiment analysis to provide personalized music suggestions.

## Project Structure

```
├── frontend/                 # React TypeScript frontend
├── backend/                  # Node.js Express TypeScript backend
├── ml-service/              # Python Flask ML service
│   ├── facial-recognition/  # Facial emotion detection
│   └── sentiment-analysis/  # Text sentiment analysis
└── docs/                    # Documentation
```

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Backend**: Node.js, Express, TypeScript
- **ML Service**: Python, Flask, OpenCV, TensorFlow
- **Database**: MongoDB (configured in backend)

Refer to individual README files in each directory for setup instructions.
